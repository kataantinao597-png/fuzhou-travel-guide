Before You Go

行前准备与交通指南

福州城区小巧而精致，住宿和交通选对了，整趟旅行就成功了一半。

🏨

住宿首选：东街口 / 南门兜

这里是地铁1号线与2号线交汇处，步行可达三坊七巷（约5分钟）和西湖公园（约10分钟）。周边夜宵选择极多，是福州老城区的心脏地带。

🧭导航至此
🚇

市内交通

下载"e福州"APP或直接使用微信/支付宝乘车码。福州地铁覆盖核心景区，公交线路四通八达。共享单车（美团/哈啰）是短途代步神器。

☀️

最佳时节与天气

福州属亚热带季风气候。10-12月秋高气爽，是最佳旅行季节。夏季（6-9月）炎热潮湿，务必防晒补水。冬季温和，适合citywalk。

🎒 行囊清单

👟

舒适运动鞋

福道钢架步道 + 烟台山爬坡，全程步行量大，一双好鞋决定旅行质量。

🧴

防晒三件套

防晒霜 + 遮阳帽 + 墨镜。福州日照强烈，即便是秋冬也需要防晒。

🔋

充电宝

一整天在外拍照、导航、扫码，电量消耗极快。建议携带10000mAh以上。

💊

常用药品

清凉油/风油精（防蚊虫）、藿香正气水（防中暑）、肠胃药（吃小吃多）。

Day One
古韵与绿意
三坊七巷 → 福道 → 青年会 · 闽江夜景
上午 · 约3小时 · 🚇 东街口站C口出

🏛️ 三坊七巷 —— 半部中国近代史

三坊七巷被誉为"明清建筑博物馆"，占地约40公顷，保留200余座明清古建筑。林则徐、严复、冰心、林觉民等名人皆出于此。

游玩要点：从南后街（主街）进入，但千万不要只在主街逛！真正的精华藏在两侧的坊巷里——

推荐深度路线：郎官巷 → 黄巷 → 塔巷 → 安泰河畔

郎官巷：严复故居（门票¥15），二梅书屋。
黄巷：小黄楼（门票¥20），精致的清代私家园林，雕梁画栋，移步换景。
塔巷：遍布文创小店和手工坊，适合淘伴手礼。
安泰河畔：沿河散步，江南水乡韵味十足。一定要找到那棵网红❤️ 爱心树拍照！

🥟 同利肉燕（南后街店）

非物质文化遗产。燕皮用猪后腿肉捶打万次而成，Q弹脆爽。推荐：肉燕+鱼丸双拼，¥15-20。

🍢 永和鱼丸（南后街店）

百年老字号。海鳗鱼肉手工打制，个头大、内馅鲜美。推荐大鱼丸，¥12/碗。

🥜 花生汤 + 芋泥

福州经典甜品CP。花生汤甜而不腻、入口即化；芋泥绵密香滑，淋上猪油更是一绝。¥8-12。

🌿 福道 —— 空中森林步道

中国首条钢结构镂空悬空步道，全长约19公里，曾获DFA亚洲设计大奖。钢架镂空设计让阳光和雨水穿透而下，走在上面仿佛漂浮于森林树冠之间。

精华路段：3号口（梅峰山地公园）进 → 5号口（金牛山体育公园）出，约3公里，坡度平缓，步行约1.5-2小时。

最佳时间：下午3-5点，光线穿过树冠，光影斑驳，拍照绝美

亮点体验：俯瞰福州城市天际线与山峦交叠；天然氧吧——森林覆盖率超90%；多个观景平台适合休息和拍照；傍晚时分金色光线洒落，是摄影爱好者的天堂。

🧭高德导航 🗺️百度导航
🚕 打车约18元（15分钟）/ 🚌 公交至青年广场站
傍晚至晚上 · 约2小时 · 🌃 闽江之心

🌉 青年广场 & 闽江夜景

青年会红砖楼：始建于1916年的百年红砖建筑，曾是福州基督教青年会会所。如今是复古风拍照圣地，红砖拱廊极具胶片感。

闽江之心：沿闽江岸线漫步，吹着江风看日落。夜晚两岸灯光亮起，江水倒映璀璨灯火。

中洲岛：位于闽江中央的欧式建筑群，被网友称为福州版"布达佩斯大饭店"。夜晚灯光亮起，粉色城堡倒映江面，极为震撼。

🍜 达明美食街

最热闹的美食夜市。推荐：锅边糊、海蛎饼、芋粿、葱肉饼。人均¥30-50吃到撑。

🥢 老福洲徐记（达明路店）

正统福州菜馆。必点：荔枝肉（¥38）、南煎肝（¥32）、芋泥（¥12）。人均¥60-80。

🍲 老药洲街小吃

没吃饱续摊首选。拌面+扁肉（¥10）、卤味，福州人最爱的宵夜搭配。

Day Two
鼓山登高与船政风云
鼓山涌泉寺 → 船政文化景区 → 上下杭
上午 · 约3-4小时 · 🚇 2号线鼓山站

⛰️ 鼓山 —— 福州的城市阳台

鼓山是福州最负盛名的地标，海拔925米，因山顶有巨石如鼓而得名。山中有千年古刹涌泉寺（门票¥40），始建于908年，被誉为"闽刹之冠"，寺内供奉的佛祖极其灵验，香火鼎盛。

上山方式二选一：
🚡 缆车轻松游：单程¥50/往返¥70，18分钟直达山顶，俯瞰福州全景。
🥾 古道登山游：从山脚沿"鼓山十八景"古道上行，约1-1.5小时，沿途可欣赏摩崖石刻——鼓山有历代文人题刻300余处，是"东南碑林"。

推荐：缆车上山（省体力）→ 涌泉寺 → 摩崖石刻 → 步行下山（感受古道）

新推体验：景区近期推出"摩崖石刻专线"和"自然探索专线"，甚至有拓印体验活动，可亲手拓制百年石刻。山顶观景台可眺望闽江入海口和福州城市全貌，天气好时能看到马尾港。

🚕 打车约30元（20分钟）→ 马尾区船政文化景区
下午 · 约2-3小时 · 🚕 马尾区

⚓ 船政文化景区 —— 中国近代海军摇篮

马尾船政是中国近代工业的发祥地之一。1866年左宗棠在此创办福州船政局，建造了中国第一艘千吨级蒸汽军舰"万年青"号。这里红砖厂房、百年轮机车间与闽江码头交织，工业风浓郁，拍照极为出片。

船政文化博物馆 → 轮机车间 → 绘事院 → 罗星塔

船政文化博物馆（免费）：全面展示中国近代海军、造船、航空工业发展历程，馆藏丰富。
轮机车间与绘事院：百年红砖建筑群，巨大的钢架结构、老式机床，是工业风摄影圣地。
《最忆船政》沉浸式演出：大型实景演艺，以光影和戏剧再现船政风云，强烈推荐（需提前购票，¥128起）。
罗星塔：马尾地标，始建于宋代的七层石塔，矗立闽江畔，被誉为"中国塔"。

🦐 马尾海鲜排档

马尾靠海，海鲜极其新鲜。推荐：白灼虾、蒜蓉生蚝、椒盐皮皮虾。人均¥60-80。

🍜 马尾鱼面

马尾本地特色，用鱼肉打成面条，Q弹鲜美。¥15-20/碗。

傍晚 · 约2-3小时 · 🚇 达道站

🏮 上下杭 —— 百年商埠的旧梦新颜

上下杭曾是福州最繁华的商业中心，"杭"即航道。宋元时期即为闽江下游重要港口。修复后的古厝沿河排开，红灯笼映衬白墙黛瓦，比三坊七巷更安静、更本地。

必逛：
三捷河两岸：沿河步道是上下杭最美的部分，古桥、石阶、红灯笼倒映水中，夜景尤其出片。
隆平路/下杭路：老字号商铺、非遗展示馆、茶空间——福州是茉莉花茶发源地，可以体验茉莉花茶制作。

🍲 依土捞化

福州捞化代表店。自选猪杂/海鲜+高汤+兴化粉。推荐：猪腰+海蛎+兴化粉，¥18-25。

🥟 耳聋伯元宵

咸口元宵的传奇。外皮糯米软糯，内馅猪肉咸香。福州人的元宵是咸的！¥12/碗。

🍡 轮机糕 / 马蹄糕

街边随处可见。轮机糕松软微甜，马蹄糕Q弹清爽，¥5-8的街头快乐。

Day Three
古镇清幽与万国风情
螺洲古镇 → 烟台山商业漫步街区
上午 · 约2-3小时 · 🚕 从市区打车约25分钟

🏘️ 螺洲古镇 —— 被时光遗忘的帝师之乡

螺洲古镇位于福州南台岛南端，因"田螺姑娘"的传说而得名。相比三坊七巷的热闹，这里几乎没有商业化气息，保留了原汁原味的闽东乡村风貌，是福州最宁静的古镇。

必看：

陈氏五楼 → 螺洲孔庙 → 吴石故居 → 螺洲江滨

陈氏五楼（陈宝琛故居）：清末帝师陈宝琛的私家园林，五座楼房依地势而建，中西合璧风格独特。门票¥15。
螺洲孔庙：福州现存最完整的乡镇级孔庙，古榕参天，环境清幽。
吴石故居："密使一号"吴石将军故里，了解那段惊心动魄的谍战历史。
螺洲江滨：沿乌龙江漫步，看渔民撒网、白鹭翩飞，感受真正的福州水乡生活。

🐚 螺洲河鲜

镇上的小馆子沿江而开，推荐：清蒸白刀鱼、蚬子汤。人均¥40-60，鲜到掉眉毛。

🍘 螺洲光饼

福州传统面点，外酥内软，夹上糟菜或红糟肉，¥3-5/个。

🏰 烟台山 —— 万国建筑博物馆

鸦片战争后福州列为通商口岸，17国在烟台山设领事馆。百年老洋房与文艺小店交相辉映，适合漫无目的地闲逛，每一步都可能遇见惊喜。

亭下路 → 石厝教堂 → 烟台山公园 → 马厂街

亭下路：核心商业街，红砖老建筑里藏着独立咖啡馆、古着店、香氛实验室、手作工坊。每家店的门头都值得拍。
石厝教堂：哥特式花岗岩教堂，银杏季（11-12月）满树金黄，福州最美秋景Top 3。
烟台山公园：山顶俯瞰闽江、中洲岛和仓山老洋房全景——日暮时分尤为壮美。
马厂街：避开人群的安静老街巷，感受原汁原味的仓山本地生活。

☕ 崔记咖啡

烟台山老牌咖啡馆，藏在老洋房里，手冲单品豆品质极佳。¥25-40。

🍽️ 烟台山Bistro

多家西餐小馆集中在亭下路一带，Brunch和晚餐环境都很棒。人均¥80-120。

🌊 连江/罗源 —— 福州人的后海花园

如果你有多一天时间，强烈推荐去福州周边的海岸线。连江和罗源距离福州市区约60-80公里，是本地人周末看海、吃海鲜的首选目的地。

推荐路线：连江滩涂 → 罗源湾海洋世界 → 贵安欢乐世界 → 海鲜大餐

连江滩涂赶海：退潮时跟着当地渔民下滩涂，挖蛤蜊、捉螃蟹、捡海蛎。建议穿短裤+拖鞋，带个小桶。赶海最佳时间需查潮汐表。

罗源湾海洋世界（刚升级）：福建最大的海洋主题乐园，拥有白鲸、海豚表演馆和亚洲最长的海底隧道。门票¥180，适合亲子游。

贵安欢乐世界：大型游乐场+温泉度假村，可泡温泉放松旅途疲劳。温泉门票¥168起。

海鲜大餐：连江是福建著名渔港，路边大排档的海鲜都是当天捕捞。推荐：白灼九节虾、清蒸石斑鱼、海蛎煎、鱼丸汤。人均¥80-120吃到撑。

🦀 连江海鲜大排档

码头边的排档最新鲜。推荐：白灼虾、椒盐濑尿虾、海蛎煎。人均¥80-120。

🐟 罗源鱼丸

罗源的鱼丸比市区的更大更弹，全是当天捕捞的海鱼制作。¥15/碗。

🍶 连江鱼面

用鱼肉打成面条，比普通面条鲜美百倍。连江特产，市区很难吃到正宗的。

Travel Tips

行程贴士

让你的福州之行更加顺畅的小秘密。

☀️

防晒与补水

福州夏季长且炎热（5-10月），体感温度常超35°C。务必涂防晒、戴帽子，每天至少喝2L水。

👟

务必穿运动鞋

福道是镂空钢架结构，穿高跟鞋或薄底鞋非常不舒服。烟台山坡道多，上下杭石板路不平——舒适运动鞋是刚需。

🕐

老字号营业时间

很多老店下午2-4点休息，或当天备货卖完即关门。建议上午11点前或下午5点后前往。

📱

必备APP

"e福州"扫码乘地铁/公交享9折；"大众点评"看餐厅真实评价；"高德地图"步行导航比百度更准。

🍵

茉莉花茶伴手礼

福州是世界茉莉花茶发源地。上下杭和南后街茶庄可试饮后购买，推荐"春伦"或"中莉"品牌，¥50-150/罐。

🗣️

语言与沟通

福州话（闽东语）非常难懂，但市区内普通话完全通用。一句"丫霸"（很棒）可以拉近距离！
